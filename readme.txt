Para baixar o projeto Sistema Auta de SOuza, basta ir no endereço:

https://github.com/pgabrielmorais/Projeto_Sistema_Auta_de_Souza

e dar um "git clone" no projeto, ou realizar seu download.

Após isso, basta  unzipar a pasta "htdocs.rar" e substituir pela pasta 
localizada no endereço "xampp".

Para acessar as configurações, telas e códigos do sistema, basta ir na pasta
"application"

Na pasta "application/views", se encontra os conteúdos das páginas. Incluindo 
footers e headers, erros e outras telas. Lembrando que footers e headers foram 
divididos entre usuarios e admins, a princípio.

Na pasta "application/controllers", se encontra os controllers (declarações e métodos
das classes a serem utilizadas no sistema). É possível realizar inclusive buscas em bancos 
de dados.

Na pasta "assets" é onde se encontra o bootstrap e seus recursos. Os arquivos
.css se encontram na pasta "assets/bootstrap-4.3.1/css" (lembrando que para 
utilizar no sistema, é necessário chamar o arquivo no código).
